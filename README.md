## ⌨️Installation

1. Go to `Extensions`
2. Search for `Lotus Flat Theme` or go [here](https://marketplace.visualstudio.com/items?itemName=alewtschuk.lotus-flat-dark-theme) and click `Install`
3. Select `Color Theme` and choose `Lotus Flat Dark`
